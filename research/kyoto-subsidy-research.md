## 1. 京都市デジタル化推進プロジェクト（令和7年度）

### 概要

デジタル技術を活用し、持続可能な経営に向けた業務効率化や生産性向上のためデジタル化に取り組む**京都市内の中小事業者等**を対象に、ITの専門家（ITコーディネータ）による経営課題・業務課題の分析、課題解決に向けたデジタル化計画の検討・ブラッシュアップからシステム導入までの一連の事業を支援する制度。

### 支援枠（2区分）

#### ア. デジタル導入枠

| 項目 | 内容 |
|------|------|
| **対象** | ITツールを活用できていない、デジタル化初期段階にある中小企業等 |
| **補助上限額** | **40万円** |
| **補助率** | **5分の4以内（80%）** |
| **専門家派遣** | 最低2回、最大4回 |
| **ハードウェア制限** | 総額15万円以内 |
| **1次採択数** | 100社 |

#### イ. デジタル展開枠

| 項目 | 内容 |
|------|------|
| **対象** | ITツールの導入等により、既にデジタル化に取り組んでいる中小企業等 |
| **補助上限額** | **100万円** |
| **補助率** | **3分の2以内（約66%）** |
| **ハードウェア制限** | 1台あたり15万円以内 |
| **1次採択数** | 100社 |

### 補助対象経費

- ソフトウェア購入費
- システム構築費
- 導入関連経費
- クラウド利用料
- 機械設備導入費（ハードウェアは上記制限あり）

### 対象者の要件

- 京都市内に主たる事務所又は事業拠点を有する中小企業等
- 又は、主たる事業所を京都市内に設けている中小企業等で構成する団体
- **令和7年3月26日時点で開業又は設立後1年未満の者は対象外**

### 募集スケジュール

| 募集回 | 期間 | 状態 |
|--------|------|------|
| 1次募集 | 2025年3月26日（水）～ 4月25日（金）17:00必着 | **終了** |
| 2次募集 | 2025年5月12日（月）～ 6月13日（金） | **終了** |

※2次募集の支援件数: 80件程度（デジタル導入枠50件、デジタル展開枠30件）

### 手続きの流れ

1. 申請
2. 審査・採択事業者決定
3. 専門家派遣（ITコーディネータ）
4. 交付申請
5. 審査・交付決定
6. 事業実施
7. 実績報告
8. 完了検査・額の確定
9. 補助金請求・受領

## 2. 京都市DXモデル構築プロジェクト

### 概要

京都市内の中小企業等におけるDX戦略の構築から実践までを**2年間にわたって**一貫支援するプロジェクト。京都市と京都高度技術研究所アステム（ASTEM）が運営。

### 支援内容

- DX専門家の派遣によるDX戦略策定支援
- 国の「DX認定制度」の申請支援
- ITツール導入経費の補助

### 採択規模

| 区分 | 社数 | 支援内容 |
|------|------|----------|
| 採択事業者 | **5社** | 専門家派遣＋補助金支援 |
| 追加採択事業者 | **5社** | 専門家派遣によるDX戦略の構築支援のみ |
| **合計** | **10社** | - |

### DX認定制度との連携

DX認定を取得すると、以下の公的支援を受けることが可能:
- 人材育成に対する支援措置
- 日本政策金融公庫からの金利優遇

### 関連プログラム: デジタル化・DX実践講座

| 項目 | 内容 |
|------|------|
| 対象 | 京都市内に本社を有する中小企業のデジタル化・DX推進責任者・担当者 |
| 定員 | 20名（1企業1名まで） |
| 内容 | 全5回の講座、最終回で自社の「DXプラン」を発表 |
| 参加費 | **無料** |

### 参考リンク

- [京都市DXモデル構築プロジェクト（ASTEM）](https://www.astem.or.jp/entre/dx-model)
- [京都府のデジタル政策・DX推進](https://www.pref.kyoto.jp/digital/index.html)

## 3. 京都府中小企業賃上げ実現緊急支援事業費補助金

### 概要

持続的な賃上げの実現に向けた事業計画に基づき、中長期的な収益力向上につながる経営改革・経営改善に取り組む**京都府内の中小企業者**を支援する補助金。（公財）京都産業21が運営。

### ★ 現在募集中 ★

| 項目 | 内容 |
|------|------|
| **募集期間** | **2026年2月4日（水）～ 3月16日（月）17:00必着** |
| **申請方法** | 交付申請書一式を郵送 |

### 補助対象事業者（要件）

以下の**すべて**を満たす必要あり:

1. **京都府内に事業所を有する中小企業者等**
2. 京都府の「**子育て環境日本一に向けた職場づくり行動宣言**」を行う者
3. **令和8年（2026年）2月1日から同年8月31日の間**に所定の賃上げを行う者（「賃上げ要件」）

※賃上げ要件の具体的なパーセンテージは公式サイトの表に記載（PDF/申請の手引き参照）

### 補助金額・補助率

| 対象経費 | 補助率 | 上限額 |
|----------|--------|--------|
| 経営基盤強化に資する設備投資等（機器・設備の導入、経営コンサルティング、人材育成等） | **2分の1** | **500万円** |
| 就業規則等の整備・改正 | **2分の1** | **10万円** |

### 対象経費の例

- 機器・設備の導入費用
- 経営コンサルティング費用
- 人材育成費用
- 就業規則等の整備・改正費用

### オンライン説明会

| 項目 | 内容 |
|------|------|
| 日時 | **令和8年（2026年）2月10日（火）13:30 ～ 14:00** |
| 形式 | ZOOMウェビナー |
| 内容 | 補助金概要、申請のポイント |

### 予算規模

京都府は事業費として**約2.8億円**を計上。最低賃金の引き上げ額を上回る賃上げをした企業を対象に上限500万円まで補助。

## 4. 関連補助金: 京都府中小企業経営基盤強化推進事業費補助金・奨励金

### 概要

賃上げが可能となる経営基盤強化を図るとともに、誰もが働きやすい職場環境の整備を促進するため、賃上げの原資となる即効的な経費削減効果に資する事業に取り組む京都府内の中小企業者に対して、その経費の一部を支援するもの。

### 補助金

| 項目 | 内容 |
|------|------|
| **対象経費** | 経営基盤強化に資する設備投資等（機器・設備の導入、経営コンサルティング、人材育成等） |
| **補助率** | 2分の1 |
| **上限額** | **500万円** |

| 項目 | 内容 |
|------|------|
| **対象経費** | 就業規則等の整備・改正 |
| **補助率** | 2分の1 |
| **上限額** | **10万円** |

### 奨励金

| 項目 | 内容 |
|------|------|
| **対象** | 令和7年6月14日以降に交付申請を行い、令和8年2月10日までに国（厚生労働省）の業務改善助成金の交付額確定・支給決定通知書を受けた者 |
| **補助率** | 国助成金の対象経費支出済額から国助成額を除いた額の2分の1 |
| **上限額** | **500万円** |

> **注意**: 補助金と奨励金を合わせて**500万円が上限**

- [京都府公式ページ](https://www.pref.kyoto.jp/rosei/keiei/gaiyou.html)
