# 3棟ヴィラ宿泊施設 財務モデル設計書

## 1. シート構成一覧

| シート名 | 目的 | 主な列・項目 |
|---------|------|-------------|
| **Inputs** | 全前提条件の入力シート | カテゴリ、項目名、値、単位、備考 |
| **Monthly** | 月次PL・CF（36ヶ月） | 月番号、年月、売上、費用、EBITDA、借入返済、FCF |
| **Annual** | 年次PL・CF（10年） | 年度、売上、費用、減価償却、営業利益、FCF、累積CF |
| **Dashboard** | KPIサマリー | 指標名、値、単位、評価 |
| **Report** | 文章レポート自動生成 | セクション、本文（数値参照で動的生成） |

## 3. 主要数式一覧（Google Sheets用）

### 3-1. Inputs シート

```
【固定費合計】
C33: =SUM(C27:C32)
```

---

### 3-2. Monthly シート

#### A列〜D列：基本情報

```
【月番号】
A2: =1
A3: =A2+1  （A37まで下方コピー）

【年月】
B2: =DATE(2025,1,1)
B3: =EDATE(B2,1)  （B37まで下方コピー）
※表示形式: yyyy/mm

【年度】
C2: =CEILING(A2/12,1)

【稼働率（インフレ反映済ADRと連動）】
D2: =IF(C2=1,Inputs!$C$17/100,IF(C2=2,Inputs!$C$18/100,Inputs!$C$19/100))
```

#### E列〜G列：売上計算

```
【インフレ調整済ADR（補助計算、隠し列でもOK）】
※年度ごとにインフレ率を累積適用
インフレ係数 = (1+Inputs!$C$21/100)^(C2-1)

【宿泊売上】
E2: =Inputs!$C$4 * D2 * (365/12) * Inputs!$C$16 * (1+Inputs!$C$21/100)^(C2-1)

【付帯売上】
F2: =E2 * Inputs!$C$20/100

【売上合計】
G2: =E2+F2
```

#### H列〜J列：費用計算

```
【変動費】
H2: =G2 * Inputs!$C$24/100

【固定費（インフレ調整）】
I2: =Inputs!$C$33 * (1+Inputs!$C$21/100)^(C2-1)

【費用合計】
J2: =H2+I2
```

#### K列〜M列：利益計算

```
【EBITDA】
K2: =G2-J2

【減価償却（月次按分）】
L2: =(Inputs!$C$36/Inputs!$C$37 + Inputs!$C$38/Inputs!$C$39 + Inputs!$C$40/Inputs!$C$41)/12

【営業利益】
M2: =K2-L2
```

#### N列〜O列：借入返済計算

```
【月利】
月利 = Inputs!$C$10/100/12

【返済回数】
返済回数 = Inputs!$C$11*12

【元利均等の場合の月額返済額（PMT関数）】
月額返済 = PMT(Inputs!$C$10/100/12, Inputs!$C$11*12, -Inputs!$C$7)

【支払利息（元利均等）】
N2: =IF(A2<=Inputs!$C$12,
       Inputs!$C$7*Inputs!$C$10/100/12,
       IF(A2>Inputs!$C$11*12+Inputs!$C$12, 0,
         IPMT(Inputs!$C$10/100/12, A2-Inputs!$C$12, Inputs!$C$11*12, -Inputs!$C$7)))

【元本返済（元利均等）】
O2: =IF(A2<=Inputs!$C$12,
       0,
       IF(A2>Inputs!$C$11*12+Inputs!$C$12, 0,
         PPMT(Inputs!$C$10/100/12, A2-Inputs!$C$12, Inputs!$C$11*12, -Inputs!$C$7)))
```

#### P列〜T列：最終損益・CF

```
【税引前利益】
P2: =M2-N2

【更新留保】
Q2: =G2*Inputs!$C$44/100

【FCF（フリーキャッシュフロー）】
R2: =K2-N2-O2-Q2

【累積FCF】
S2: =-Inputs!$C$6+R2
S3: =S2+R3  （以下コピー）

【DSCR（月次）】
T2: =IFERROR((K2-Q2)/(N2+O2),"-")
```

### 3-3. Annual シート

#### 基本集計（月次から年次へ）

```
【年度】
A2: =1
A3: =A2+1  （A11まで）

【宿泊売上】
B2: =SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$E$2:$E$37)

【付帯売上】
C2: =SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$F$2:$F$37)

【売上合計】
D2: =B2+C2

【変動費】
E2: =SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$H$2:$H$37)

【固定費】
F2: =SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$I$2:$I$37)

【EBITDA】
G2: =D2-E2-F2
```

#### 4年目以降の予測（月次データがない年）

```
【4年目以降の売上（A5以降）】
B5: =Inputs!$C$4 * (Inputs!$C$19/100) * 365 * Inputs!$C$16 * (1+Inputs!$C$21/100)^(A5-1)

【4年目以降の付帯売上】
C5: =B5 * Inputs!$C$20/100

【4年目以降の変動費】
E5: =D5 * Inputs!$C$24/100

【4年目以降の固定費】
F5: =Inputs!$C$33 * 12 * (1+Inputs!$C$21/100)^(A5-1)
```

#### 減価償却・借入返済

```
【減価償却（年次）】
H2: =Inputs!$C$36/Inputs!$C$37 + Inputs!$C$38/Inputs!$C$39 + Inputs!$C$40/Inputs!$C$41

※耐用年数経過後はゼロにする条件を追加：
H2: =IF(A2<=Inputs!$C$37,Inputs!$C$36/Inputs!$C$37,0)
    +IF(A2<=Inputs!$C$39,Inputs!$C$38/Inputs!$C$39,0)
    +IF(A2<=Inputs!$C$41,Inputs!$C$40/Inputs!$C$41,0)

【支払利息（年次集計）】
I2: =IF(A2<=3, SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$N$2:$N$37),
      IF(A2>Inputs!$C$11,0,
        SUMPRODUCT((ROW(INDIRECT("1:"&12))+((A2-1)*12)<=Inputs!$C$11*12)*
        IPMT(Inputs!$C$10/100/12,ROW(INDIRECT("1:"&12))+((A2-1)*12),
             Inputs!$C$11*12,-Inputs!$C$7))))

【簡易版：年次利息計算】
I2: =IF(A2<=3,
       SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$N$2:$N$37),
       MAX(0, (Inputs!$C$7 - PPMT_CUMULATIVE) * Inputs!$C$10/100))

※より簡易な近似式：
I2: =IFERROR(
      IF(A2*12>Inputs!$C$11*12,0,
        12*IPMT(Inputs!$C$10/100/12, ((A2-1)*12)+6, Inputs!$C$11*12, -Inputs!$C$7)),
      0)

【元本返済（年次）】
J2: =IF(A2<=3,
       SUMIF(Monthly!$C$2:$C$37,A2,Monthly!$O$2:$O$37),
       IF(A2>Inputs!$C$11, 0,
         12*PPMT(Inputs!$C$10/100/12, ((A2-1)*12)+6, Inputs!$C$11*12, -Inputs!$C$7)))
```

#### 最終損益・FCF

```
【営業利益】
=G2-H2

【税引前利益】
K2: =G2-H2-I2

【法人税等（実効税率30%想定）】
L2: =MAX(0, K2*0.30)

【当期純利益】
M2: =K2-L2

【更新留保】
N2: =D2*Inputs!$C$44/100

【FCF】
O2: =G2-I2-J2-N2

【累積FCF】
P2: =-Inputs!$C$6+O2
P3: =P2+O3  （以下コピー）

【DSCR】
Q2: =IFERROR((G2-N2)/(I2+J2), "-")

【投資残高（未回収額）】
R2: =Inputs!$C$6-P2
※マイナスになったら回収完了
```

### 3-4. Dashboard シート

```
【10年累計売上】
B4: =SUM(Annual!$D$2:$D$11)

【10年累計EBITDA】
B5: =SUM(Annual!$G$2:$G$11)

【EBITDA率（3年目）】
B6: =Annual!$G$4/Annual!$D$4

【投資回収年数（Payback Period）】
B9: =IFERROR(
      MATCH(TRUE,Annual!$P$2:$P$11>=0,0)+
      (0-INDEX(Annual!$P$2:$P$11,MATCH(TRUE,Annual!$P$2:$P$11>=0,0)-1))/
      (INDEX(Annual!$P$2:$P$11,MATCH(TRUE,Annual!$P$2:$P$11>=0,0))-
       INDEX(Annual!$P$2:$P$11,MATCH(TRUE,Annual!$P$2:$P$11>=0,0)-1)),
      "10年超")

※簡易版：
B9: =COUNTIF(Annual!$P$2:$P$11,"<0")+1

【IRR（10年）】
B10: =IRR({-Inputs!$C$6,Annual!$O$2,Annual!$O$3,Annual!$O$4,Annual!$O$5,
           Annual!$O$6,Annual!$O$7,Annual!$O$8,Annual!$O$9,Annual!$O$10,Annual!$O$11})

※配列が使えない場合、補助列を作成してIRR関数を使用

【NPV（割引率5%）】
B11: =NPV(0.05,Annual!$O$2:$O$11)-Inputs!$C$6

【平均DSCR】
B14: =AVERAGEIF(Annual!$Q$2:$Q$11,">0")

【最低DSCR】
B15: =MINIFS(Annual!$Q$2:$Q$11,Annual!$Q$2:$Q$11,">0")

【損益分岐稼働率】
※固定費÷（ADR×棟数×365×(1-変動費率)）
B16: =(Inputs!$C$33*12)/(Inputs!$C$16*Inputs!$C$4*365*(1-Inputs!$C$24/100))

【累積FCF（10年）】
B19: =Annual!$P$11

【年間平均FCF】
B20: =AVERAGE(Annual!$O$2:$O$11)
```

## 5. スプレッドシート作成手順

### Step 1: シートを5つ作成
1. Inputs
2. Monthly
3. Annual
4. Dashboard
5. Report

### Step 2: Inputsシートに値を入力
- 上記の配置に従い、カテゴリ・項目名・値・単位を入力
- C33に固定費合計の数式を設定

### Step 3: 名前付き範囲を設定（推奨）
- データ → 名前付き範囲 から主要セルに名前を設定
- 数式が読みやすくなり、メンテナンス性向上

### Step 4: Monthlyシートを構築
- A1〜T1にヘッダーを入力
- A2〜A37に月番号1〜36を設定
- 各列の数式をA2行に入力し、37行目までコピー

### Step 5: Annualシートを構築
- A1〜R1にヘッダーを入力
- 1〜3年目はMonthlyからSUMIF集計
- 4〜10年目は直接計算式を設定

### Step 6: Dashboardシートを構築
- 各KPIの計算式を設定
- 表示形式を整える（通貨、パーセント等）

### Step 7: Reportシートを構築
- CONCATENATE数式で文章を構築
- セル幅を広げ、折り返し表示を設定

### Step 8: 検証
- 手計算と照合して数式の正確性を確認
- 入力値を変更し、連動して更新されることを確認

## 6. 補足：Google Sheets特有の注意点

1. **配列数式**: `{}`で配列を作成。IRR関数で使用
2. **IFERROR**: エラー時の表示を制御。DSCR計算で使用
3. **TEXT関数**: 数値を文字列にフォーマット。レポートで使用
4. **CONCATENATE / &**: 文字列連結。レポートで使用
5. **CHAR(10)**: 改行コード。レポートの段落で使用
6. **PMT/IPMT/PPMT**: ローン計算の標準関数
7. **名前付き範囲**: 絶対参照より推奨。可読性向上
